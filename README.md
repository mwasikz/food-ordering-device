## Food Ordering Device

:round_pushpin: A food ordering touchscreen device for <a href="https://github.com/mwasikz/robo-cafe-rms">Robo Cafe</a>. The device is created using Arduino and TFT Touch Screen Module. It is also connected to the database of <a href="https://github.com/mwasikz/robo-cafe-rms">Robo Cafe Admin Dashboard</a>. Each time a order is placed a notification will show up in the Admin Dashboard containing the order information.
